Pergunta: Me ensine, na prática, dizendo o que exatamente eu deveria digitar no prompt para usar este projeto em uma pasta de um projeto meu de nome intranet-legado. Considere que a pasta está no caminho fake /meu/projeto/intranet-legado

Resposta: Com certeza! Vamos simular isso na prática.

### Pré-requisito: Configurando o Ambiente

Antes de executar qualquer comando, você precisa preparar seu projeto para que eu possa encontrar e utilizar os agentes e comandos customizados. Siga estes passos **dentro da raiz do seu projeto** (`/meu/projeto/intranet-legado`):

1.  **Crie a pasta de configuração:**
    ```bash
    mkdir -p .claude/agents .claude/commands
    ```

2.  **Copie os arquivos dos agentes e comandos:**
    *   Copie os arquivos de `4-prompts-e-workflow-de-agentes/agents/` para a pasta `.claude/agents/` que você acabou de criar.
    *   Copie o arquivo de `4-prompts-e-workflow-de-agentes/commands/` para a pasta `.claude/commands/`.

Seu projeto `/meu/projeto/intranet-legado` deverá ter a seguinte estrutura:

```
/meu/projeto/intranet-legado/
|-- .claude/
|   |-- agents/
|   |   |-- architectural-analyzer.md
|   |   |-- component-deep-analyzer.md
|   |   |-- dependency-auditor.md
|   |   |-- orchestrator.md
|   |-- commands/
|       |-- run-project-state-full-report.md
|-- (outros arquivos e pastas do seu projeto)
```

Com essa estrutura, eu serei capaz de encontrar e executar o workflow definido.

### Executando o Relatório

Agora que o ambiente está configurado, para executar o relatório completo do seu projeto "intranet-legado", você usaria o comando principal `run-project-state-full-report` e especificaria a pasta do projeto com o argumento `--project-folder`.

Você digitaria exatamente isto no seu prompt:

```
/run-project-state-full-report --project-folder=/meu/projeto/intranet-legado
```

Ou o comando abaixo caso já esteja na pasta do projeto:
```
/run-project-state-full-report --ignore-folders=cache,runtime,vendor,node_modules,automacao_k8s,nginx --output-folder=analise-resultado
```

### O que Aconteceria a Seguir

Ao receber esse comando, como seu Coordenador Mestre, eu começaria a orquestrar os agentes da seguinte forma:

1.  **Fase 1: Inicialização do `orchestrator`**
    *   Eu chamaria o agente `orchestrator` para iniciar o processo, definindo `/meu/projeto/intranet-legado` como a pasta do projeto a ser analisada.
    *   O `orchestrator` criaria as pastas necessárias dentro da estrutura `docs/agents/` e o arquivo `MANIFEST.md`.
    *   Eu te informaria: "Iniciando a análise do projeto 'intranet-legado'. O agente orchestrator está preparando o ambiente e o MANIFEST.md inicial."

2.  **Fase 2: Análise Paralela (Auditoria de Dependências e Arquitetura)**
    *   Em paralelo, eu chamaria:
        *   O `dependency-auditor` para escanear seu projeto em busca de dependências.
        *   O `architectural-analyzer` para mapear a arquitetura de alto nível.
    *   Conforme cada agente completasse, eu chamaria o `orchestrator` novamente para registrar seus relatórios no `MANIFEST.md`.
    *   Eu te informaria sobre o progresso de cada um.

3.  **Fase 3: Análise Detalhada dos Componentes**
    *   Após a análise da arquitetura, eu leria o relatório para identificar os "Componentes Críticos".
    *   Para cada componente, eu iniciaria uma tarefa com o `component-deep-analyzer` (por exemplo, "Módulo de Usuários", "Módulo de Relatórios").
    *   A cada relatório de componente finalizado, o `orchestrator` registraria a informação no `MANIFEST.md`.

4.  **Fase 4: Finalização do `orchestrator`**
    *   Após todos os relatórios serem registrados, eu chamaria o `orchestrator` uma última vez para validar o `MANIFEST.md`.
    *   Eu te informaria: "Agente orchestrator finalizando o MANIFEST.md, verificando a integridade e links."

5.  **Fase 5: Geração do `README.md` Final (Meu Trabalho)**
    *   Eu leria o `MANIFEST.md` completo e geraria o arquivo `README-YYYY-MM-DD-HH:MM:SS.md` com um sumário e links para todos os relatórios.
    *   Eu te apresentaria a mensagem final: "Relatório completo do projeto 'intranet-legado' gerado com sucesso! Você pode encontrar o README final e todos os relatórios em `docs/agents/orchestrator/`."

Em resumo, após a configuração inicial da pasta `.claude`, você só precisaria digitar o comando. Eu faria o restante do trabalho de coordenação.
